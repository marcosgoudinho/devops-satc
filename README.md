# devops-satc

Projeto React gerado com Create React App.

Guia rápido:

- **Instalar dependências:** `npm install`
- **Desenvolvimento:** `npm start` (abre em `http://localhost:3000`)
- **Testes:** `npm test`
- **Build para produção:** `npm run build`

Links úteis:

- Documentação Create React App: https://create-react-app.dev/
- React: https://reactjs.org/

Pronto — edite `src/App.js` para começar.
